База знаний по backend разработке

Страницы :
Главная - сверху страницы отображает топ 3 самых просматриваемых и комментируемых поста,под топом постов отображает грид сетку из категорий, которые создаются через панель администратора
Страница категории - отображает посты которые создаются через панель администратора, каждый пост должен быть привязан к категории
Страница поста - Отображает контент поста, имеет кнопки поделиться и блок в котором можно оставить комментарий
Страница админ панели - Базовый функционал панели администратора. Удаление/редактирование постов, смена категории для поста. Модерация комментариев к постам. Редактирование название категории(удалять нельзя). Эта страница должна быть доступна по роуту /admin, не индексироваться и войти в нее можно только по паролю администратора. После входа администратор получает JWT токен который должен храниться в secure cookie на 1 час.

Также в хедере слева должен быть текст логотип (как сейчас) а справа ссылка на страницу "О проекте"
Страница о проекте - информация о том что за проект, кто создатель, ссылки на соц сети создателя. Также краткое описание функционала.
